===================
fizzbuzzn(practice)
===================
| 簡単なfizzbuzzプログラムからバージョンアップを繰り返すことで、
| pythonでのプログラムの作り方を学んでいく。

usage (v0.0.3)
--------------
apiはfizzbuzz(num)関数のみ

.. code-block:: python

    >>> from fizzbuzzn.app import fizzbuzz
    >>> fizzbuzz(1)
    1
    >>> fizzbuzz(15)
    FizzBuzz
